# Medical Insurance Premium Predictor

An end-to-end machine learning project implementing multiple linear regression from scratch using NumPy.

## Features

- Data preprocessing and feature engineering
- Manual implementation of gradient descent
- Feature scaling using standardization
- Model evaluation using RMSE, MAE, and R²
- Flask web application for premium prediction

## Tech Stack

- Python
- NumPy
- Pandas
- Flask
- HTML/CSS

## Model

The model was implemented without using scikit-learn.

## Metrics

- RMSE: 167
- R²: 0.67

pip install -r requirements.txt
python app.py
